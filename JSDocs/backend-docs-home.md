# PokeIQ Backend Documentation

This documentation site covers the Express backend for PokeIQ.

It includes:
- API route handlers
- DynamoDB integration code
- Backend utility modules
- Server bootstrap behavior

## Scope

The backend is responsible for serving API endpoints, interacting with DynamoDB, and handling operational concerns such as logging and health checks.

## Main Areas

- Server setup
- DynamoDB API routes
- DynamoDB service layer
- Utility helpers
