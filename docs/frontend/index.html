PokeIQ Frontend Documentation

This documentation site covers the React frontend for PokeIQ.

It includes:

  • Top-level application structure
  • Pages and reusable UI components
  • Quiz-related components and logic
  • Frontend utility modules

Scope

The frontend is responsible for rendering the PokeIQ user interface, managing client-side routing, and supporting quiz interactions and user flows.

Main Areas

  • App bootstrapping and routing
  • Page components
  • Quiz components
  • Drag and drop behavior
  • Frontend utilities
