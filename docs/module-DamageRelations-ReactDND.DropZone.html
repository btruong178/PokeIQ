DamageRelations-ReactDND. DropZone

new DropZone(props) → {JSX.Element}

Description:

  • DropZone is a React component that represents a drop target for draggable Pokémon types in the Damage Relations Quiz.
    Populates itself with DraggableType components based on the current state of the AnswerObject.
