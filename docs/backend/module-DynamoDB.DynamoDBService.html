DynamoDBService

DynamoDB. DynamoDBService

new DynamoDBService()

Description:

  • A service class that encapsulates interactions with AWS DynamoDB using AWS SDK v3. Provides various methods for performing CRUD operations, batch operations, and health checks on DynamoDB tables.

Examples

  { id: 1, name: 'Bulbasaur', type: 'Grass/Poison' },
  { id: 2, name: 'Ivysaur', type: 'Grass/Poison' }

  {
    id: 10,
    name: 'fire',
    damage_relations: {
      double_damage_from: [{ name: 'water', url: 'https://pokeapi.co/api/v2/type/11/' }],
      double_damage_to: [{ name: 'grass', url: 'https://pokeapi.co/api/v2/type/12/' }],
      half_damage_from: [{ name: 'fire', url: 'https://pokeapi.co/api/v2/type/10/' }],
      half_damage_to: [{ name: 'water', url: 'https://pokeapi.co/api/v2/type/11/' }],
      no_damage_from: [],
      no_damage_to: []
    }
  }
]);
